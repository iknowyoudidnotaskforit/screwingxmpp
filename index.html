This page uses Candy Chat for multi user XMPP chat. Only the tiny plugins are own work, Candy is stolen ... just used.

Read here if you want to create your own chatroom and a link for users to autojoin it.

Please note: Encryption passwords are just appended to the link (if you use one, not mandatory). So if you want to specify an encryption password for your chats, just append a "#passwordhere" to the link, where "passwordhere" is your password. ALL messages in this window, group and private, will be encrypted and decrypted using this password. Of course, all participants will have to use the same password. You could for example give your contacts a link which already has a password appended. If you create a direct link for a room, please specify the encryption password if you use one. Using an encryption password is not mandatory, but can sometimes increase security.
